个人博客创建说明 - 使用 GitHub Pages

你好！这份文档将详细指导你如何使用 GitHub Pages 免费搭建一个个人博客。这种方法非常适合已拥有 HTML 文件，并希望能够方便管理和更新内容的用户。

GitHub Pages 是一个免费服务，能够直接将你的代码仓库变成一个可访问的网站，管理和更新都非常方便。

第一步：准备工作

在开始之前，请确保你已经拥有以下工具：

  • 一个 GitHub 账号。如果还没有，请先前往 GitHub 官网 注册一个。
  • Git 软件。这是将本地文件上传到 GitHub 的必要工具。你可以从 Git 官网 下载并安装。

第二步：创建 GitHub 仓库

这个仓库将作为你的博客文件存放地，并且它的名称将直接决定你的免费域名。

  1. 登录你的 GitHub 账号。
  2. 点击右上角的 "+" 号，然后选择 "New repository"。
  3. 在 Repository name 处，输入一个特殊的名称：你的用户名.github.io。例如，如果你的 GitHub 用户名是 johndoe，那么仓库名就必须是 johndoe.github.io。
  4. 选择 "Public"，这样你的网站才能被所有人访问。
  5. 勾选 "Initialize this repository with a README"。
  6. 点击 "Create repository" 完成创建。

创建成功后，你的博客免费域名就诞生了：https://你的用户名.github.io。

第三步：上传你的 HTML 文件

现在，我们将把你的本地文件上传到刚刚创建的 GitHub 仓库。

  1. 克隆仓库到本地

    打开你电脑上的终端（Windows 可用 PowerShell 或 Git Bash），运行以下命令。请将 你的用户名 替换成你自己的。

    git clone https://github.com/你的用户名/你的用户名.github.io.git

    这个命令会在你的电脑上创建一个同名文件夹，这就是你博客的本地工作区。

  2. 整理并放入你的文件

    将你所有的博客文件（HTML 文件、CSS、图片、JavaScript 等）都复制到这个新创建的文件夹中。

    • 首页文件：你的博客主页文件（通常是 index.html）必须放在这个文件夹的根目录下。
    • 目录结构：为了方便管理，你可以为不同板块和章节创建文件夹。例如：
      • /learning (板块)
  /chapter1 (章节)
    intro.html
    lesson1.html
/portfolio (另一个板块)
  project1.html
  3. 提交并推送文件

    回到终端，进入你的仓库文件夹，然后依次运行以下命令：

    # 进入你的仓库目录
cd 你的用户名.github.io

# 添加所有新文件和更改
git add .

# 记录这次更改并添加描述
git commit -m "第一次提交博客文件"

# 将更改上传到 GitHub
git push origin main

注意： 如果你的仓库默认分支是 master，请将最后一条命令的 main 改为 master。

第四步：验证和访问

完成上述步骤后，等待几分钟，你的博客就会自动部署上线。

在浏览器中输入 https://你的用户名.github.io，你就可以看到你上传的网站了。恭喜你，你的个人博客已经成功创建！

关于目录和导航

为了实现“选择板块”和“浏览章节”的功能，你需要在你的 HTML 文件中手动构建导航系统。以下是一些示例代码，你可以参考并修改。

主页文件 (index.html) 示例

在你的主页中，你可以用链接指向不同的板块文件夹。

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的个人博客</title>
</head>
<body>
    <h1>欢迎来到我的博客！</h1>
    <nav>
        <ul>
            <li><a href="/learning/">学习笔记</a></li>
            <li><a href="/portfolio/">我的作品集</a></li>
            <!-- 你可以根据需要添加更多板块 -->
        </ul>
    </nav>
</body>
</html>

板块列表文件 (例如 learning/index.html) 示例

在每个板块的文件夹中，创建一个 index.html 文件来列出所有的章节。

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>学习笔记</title>
</head>
<body>
    <h1>学习笔记</h1>
    <nav>
        <a href="/">返回主页</a>
    </nav>
    <ul>
        <li><a href="/learning/chapter1/intro.html">第一章：入门介绍</a></li>
        <li><a href="/learning/chapter1/lesson1.html">第一课：基础知识</a></li>
        <!-- 根据你的章节内容添加更多链接 -->
    </ul>
</body>
</html>

通过这种方式，你可以用最简单、最灵活的方式管理你的博客内容。每次更新，只需重复 git add、git commit 和 git push 这三个命令即可。

希望这份详细的说明能帮助你顺利创建自己的个人博客。祝你创作愉快！
